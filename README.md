This notebook implements a multi-species SEIRS model with individuals migrating over patches of land. The model was taken from "A multi-species epidemic model with spatial dynamics" by JULIEN ARINO (2005) and in particular the approach used by Tennant et al. in "Modelling the persistence and control of Rift Valley fever virus in a spatially heterogeneous landscape".

This simple model with an algebraic approach, is able to describe how the epidemics diffuse in a migrating population, compute R0 and evaluate wether the disease is endemic or not.
